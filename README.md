# IE7275-Project
Predictive Modelling on Building Energy Star Scores - A Data Mining Project

## Purpose
This project will demonstrate the applications of data mining principles and processes in a climate and energy analysis context. Regression and classification models will be applied to a building energy dataset to predict EPA Energy Star scores and categorize countries as passing or failing Energy Star certification. Feature importance outputs from the models can be used to highlight drivers of scoring, helping building owners take a more targeted approach to energy improvements.
