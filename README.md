# IE7275-Project
Predictive Modelling on Building Energy Star Scores - A Data Mining Project

## Purpose
This project will demonstrate the applications of data mining principles and processes in a climate and energy analysis context. Regression and classification models are applied to a building energy dataset to predict EPA Energy Star scores and categorize buildings as passing or failing Energy Star certification. Feature importance outputs from the models can be used to highlight drivers of scoring, helping building owners take a more targeted approach to energy improvements. More information on the project context can be found in the Project Proposal PDF.

## Data
The dataset used for this project was obtained from Kaggle: https://www.kaggle.com/datasets/city-of-seattle/sea-building-energy-benchmarking/code. Details can be found in the Project Proposal PDF.

## Modelling and Results
Ridge and Gradient Boosting were used for the regression exercise. Logistic Regression and Random Forest were used for the classification exercise.

Reasoning as well as results and analysis are discussed in markdown cells within the python notebook file.
